# Videos Directory

This directory should contain the following videos:

## Required Videos

1. `hero-bg.mp4` - Hero section background video
   - Duration: 30-60 seconds
   - Resolution: 1920x1080px
   - Format: MP4 (H.264 codec)
   - File size: Optimize to under 5MB
   - Content: Subtle, non-distracting movement that represents your brand

2. `about-bg.mp4` - About section background video
   - Duration: 30-60 seconds
   - Resolution: 1920x1080px
   - Format: MP4 (H.264 codec)
   - File size: Optimize to under 5MB
   - Content: Professional environment or work process

## Project Showcase Videos
- `projects/project1.mp4` - Video showcase of project 1
- `projects/project2.mp4` - Video showcase of project 2
- `projects/project3.mp4` - Video showcase of project 3

## Video Guidelines
- All videos should be optimized for web playback
- Use MP4 format with H.264 codec for maximum compatibility
- Keep videos short and focused
- Include a poster image (first frame) for each video
- Provide fallback images for mobile devices
- Consider creating shorter versions for mobile devices
- Ensure videos are muted by default
- Add captions or subtitles when necessary
- Test playback performance across different devices and connections
